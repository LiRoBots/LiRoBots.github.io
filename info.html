Info

LiRo Bot is a Discord bot that you can invite to your server.

The bot contains many diffrent commands that you can use, like fun and admin commands.

There is even 3 diffrent bots in the "LiRo" series, LiRo Bot, LiRo Guard and LiRo Manager.

LiRo Bot
LiRo Guard

LiRo Bot: LiRoBot is a multipurpose fun and utility bot, made for all your fun needs.

LiRo Guard: LiRoGuard is a WIP Moderation bot. When done, will be able to assist you in all the moderation of a server!

LiRo Manager: LiRoManager is a WIP Managing bot that will only be used in this discord. It makes announcements, updates, etc.
